Embed Code for Your Dealer

Copy and paste this single line anywhere on the dealer's website — before the closing </body> tag works great.

<script src="http://localhost:3001/widget.js?id=demo" ></script>

That's it! The widget will appear as a floating chat button on the dealer's site.

Magento Integration

For Magento, add to app/design/frontend/[theme]/default/layout/default.xml:

<block type="core/text" name="lamello.widget">
  <action method="setText"><text>
    <![CDATA[<script src="http://localhost:3001/widget.js?id=demo"></script>]]>
  </text></action></block>
